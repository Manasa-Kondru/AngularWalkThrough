What is Angular?

A Front-end frame-work to build client side "SINGLE PAGE APPLICATIONS" in HTML,CSS, and JavaScript/TypeScript. It is a open-source and has component based architecture.

  • Gives our applications a clean structure
  • Includes a lot of re-usable code
  • Makes our applications more testable
